<a name="about-the-project"></a>
## Sobre o Projeto

Aplicação para controlar gastos pessoais simples e responsivo feito com REACT.

<img width="1493" alt="controle-financeiro" src="https://user-images.githubusercontent.com/105018299/193169115-50b45594-9a61-4f4e-9743-c848ae23fee0.png">

**Funcionalidades principais:**
* Controle de Gastos com Entras e Saídas de valores.
* Design Responsivo.
* Salvo em localStorage.
* Função de Deletar.
* Utilizado styled-components & react-icons.

Aplicação para contrar gastos com Entrada e Saídas de valores com design responsivo, utilizável em todos os dispositivos. Todas funcionalidades estão funcionando perfeitamente, É um projeto excelente, sendo construído com REACTJS & CSS3. Espero que goste. Clique [aqui](https://appfinanceiro.netlify.app/) para ver o projeto ao vivo na web.

<a name="getting-started"></a>
## Getting Started

<a name="clone-locally"></a>
### Clonar-localmente
- Digite essa url: [https://github.com/pauloaraujo028/controle-financeiro](https://github.com/pauloaraujo028/controle-financeiro) no seu navegador.
- Uma vez aberto, navegue até o nível superior esquerdo do projeto, um botão verde de download do código estará visível no lado direito.
- Selecione a opção de download Zip no menu suspenso.
- Quando o download estiver concluído, você poderá acessar meu projeto localmente. 

<a name="prerequisites"></a>
### Pré-requisitos
* npm
  ```sh
  npm install npm@latest -g
  ```

## Contribuição

As contribuições são o que tornam a comunidade de código aberto um lugar incrível para aprender, inspirar e criar. Quaisquer contribuições que você fizer são **muito apreciadas**.

Se você tiver uma sugestão para melhorar o projeto, faça um fork do repositório e crie uma solicitação pull. Você também pode simplesmente abrir um issue com a tag "melhoria".
Não se esqueça de dar uma estrela ao projeto! Obrigado novamente!

1. Faça o Fork do Projeto
2. Crie seu Branch (`git checkout -b feature/RecursoIncrivel`)
3. Commit suas alterações (`git commit -m 'Adicione algum recurso incrivel'`)
4. Push para o Branch (`git push origin feature/RecursoIncrivel`)
5. Abra um Pull Request
